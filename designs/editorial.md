# Editorial Style

Warm, light, editorial magazine feel. Think personal essay layout meets modern web.

## Philosophy

- Off-white background with warm undertones
- Serif headings (Fraunces) paired with clean sans body (Inter)
- Rounded corners and soft shadows
- Warm orange accent color
- Content-forward — no flashy effects, just readable hierarchy
- Cards float on white with subtle borders

## Key Properties

- **Heading font**: Fraunces (optical size serif, warm personality)
- **Body font**: Inter (clean, highly legible)
- **Background**: #FAFAF8 (warm off-white)
- **Text**: #1A1A1A (primary), #5C5C5C (secondary), #8C8C8C (tertiary)
- **Accent**: #D4760A (warm orange), gradient to #E8943D / #F2B872
- **Borders**: #EBEBEB
- **Card background**: #FFFFFF
- **Border-radius**: 12px (cards), 6-10px (smaller elements)
- **Section spacing**: 56px margin-bottom
- **Max-width**: 720px (narrow, readable column)

## Visual Characteristics

- Avatar circle with gradient and first-name initial (via `data-initial` attribute)
- Subtle gradient wash behind hero (7% opacity)
- Cards have hover shadow effect (lifts slightly)
- Pull quotes use left border accent (3px orange)
- Trait icons sit in small rounded squares with tinted background (#FFF3E0)
- Bar charts use 24px-tall rounded bars with gradient fills
- Lists use colored dot bullets (green/red)
- Fun facts sit in a warm-tinted container (#FFF3E0)
- Tags are pill-shaped with gray background
- Footer is simple centered text

## Mood

Approachable. Professional without being corporate. Like reading a well-designed personal essay — warm, clear, inviting. The serif headings add personality without pretension.
