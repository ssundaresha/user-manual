# Lottie Style

Clean, playful, modern SaaS. Inspired by Lottie/LottieFiles aesthetic.

## Philosophy

- Pure white background, maximum clarity
- Single teal accent color used consistently
- Generous border-radius (16px cards, 10px bars)
- Subtle shadows and hover animations
- DM Sans for headings (friendly, geometric)
- Centered hero with radial gradient glow
- Three-column trait cards

## Key Properties

- **Heading font**: DM Sans (geometric, friendly, modern)
- **Body font**: Inter (clean)
- **Background**: #ffffff
- **Text**: #09090b (primary), #71717b (secondary), #9f9fa9 (tertiary)
- **Accent**: #019d91 (teal), gradient to #00ddb3 (bright mint)
- **Borders**: #e4e4e7
- **Muted backgrounds**: #f4f4f5
- **Border-radius**: 16px (cards), 10px (bars/icons), 8px (week blocks)
- **Card shadows**: rgba(0,0,0,0.05) 0px 1px 2px
- **Section spacing**: 64px margin-bottom
- **Max-width**: 800px

## Visual Characteristics

- Hero is centered with role text in teal above the name
- Radial gradient glow behind hero (teal, very subtle)
- Trait cards are three-column with hover lift + shadow increase
- Trait icons use teal-tinted rounded squares
- Flags section uses colored bottom-borders on titles (teal/red)
- Bar charts use 20px rounded bars with teal gradient
- Energy cards match trait card style (16px radius, shadow)
- Pull quotes sit in gray rounded containers (no border)
- Week blocks use 8px radius
- Tags are extra-rounded pills (12px radius)
- Fun facts use gradient tinted background (teal at 5%)
- Everything feels like a modern product UI

## Mood

Friendly. Polished. Like a well-designed product page — everything is intentional, rounded, and inviting. The teal accent adds energy without being overwhelming. Feels like it could be a page inside a modern SaaS app.
